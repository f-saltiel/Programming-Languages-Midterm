Program Information

This extra program uses JavaScript to accept a player choice, generate a random computer choice, determine the winner, and update the scoreboard each round.

  • Language: JavaScript
  • Requirement: Extra Program
  • Input: Player move selection
  • Output: Round result and updated wins, losses, ties, and total rounds
