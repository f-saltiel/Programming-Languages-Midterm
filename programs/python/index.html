Program Information

This program demonstrates the imperative programming paradigm by using step-by-step Python logic to collect user input, look up component scores, calculate a composite score, and display the result.

  • Language: Python
  • Browser Tool: PyScript
  • Paradigm: Imperative
  • Input: Gender, age group, push-ups, sit-ups, run time, waist, and height
  • Output: Component scores, WHtR, total score, category, and warnings
