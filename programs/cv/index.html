Program Information

This program loads structured CV data from a JSON file, filters active entries, renders a resume in HTML, and downloads the rendered resume as a PDF.

  • Data File: cv.json
  • Requirement: Required CV Program
  • Input: Active/inactive entry selections
  • Output: Resume preview and downloadable PDF
