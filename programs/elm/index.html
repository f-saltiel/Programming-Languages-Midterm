Program Information

This program demonstrates the functional programming paradigm by using pure functions to transform distance, pace, and time inputs into calculated output values.

  • Language: Elm
  • Browser Output: Compiled JavaScript
  • Paradigm: Functional
  • Input: Distance, pace, and time values
  • Output: Converted distance, converted pace, and calculated race pace
